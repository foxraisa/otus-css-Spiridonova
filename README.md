# Лендинг компании Mod Studio
Здесь будет описание проекта + ссылка на задеплоенный сайт 

## Как установить прект

1. Скачать проект с GitHub
2. Установить зависимости с помощью команды 'npm init'
3. Открыть проект с помощью команд:
- 'npm run build-webpack' для запуска Development сборки;
- 'npm run build-prod' для запуска Production сборки.
